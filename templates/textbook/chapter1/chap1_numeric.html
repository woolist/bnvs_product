Chapter 2: 숫자형 이해하기

2.1 숫자형이란?

파이썬에서는 여러가지 숫자형을 사용할 수 있습니다. 가장 많이 사용되는 숫자형은 정수형(int)과 실수형(float)입니다. 또한 복소수형(complex)도 제공합니다.

2.2 정수형

정수형(int)는 양의 정수, 음의 정수, 0을 포함합니다. 예를 들어, 10, -7, 0 등이 있습니다.

2.2.1 정수형 만들기

1
2
a = 5
print(type(a)) # 출력: <class 'int'>
cs

2.3 실수형

실수형(float)는 소수점이 포함된 숫자를 나타냅니다. 예를 들어, 3.14, -0.7, 8.0 등이 있습니다.

2.3.1 실수형 만들기

1
2
b = 3.2
print(type(b)) # 출력: <class 'float'>
cs

2.4 복소수형

복소수형(complex)는 복소수를 나타냅니다. 복소수는 실수부와 허수부로 구성되며, 허수부는 'j' 또는 'J'를 사용하여 표시합니다.

2.4.1 복소수형 만들기

1
2
c = 1+2j
print(type(c)) # 출력: <class 'complex'>
cs

2.5 숫자형 연산

파이썬에서는 숫자형 사이의 기본적인 사칙연산을 제공합니다. 더하기, 빼기, 곱하기, 나누기 연산 외에도 몫과 나머지를 구하는 연산도 제공합니다.

2.5.1 숫자형 연산 예시

1
2
3
4
5
6
7
8
9
a = 10
b = 3

print(a - b)  # 출력: 7
print(a * b)  # 출력: 30
print(a / b)  # 출력: 3.3333333333333335
print(a // b) # 출력: 3
print(a % b)  # 출력: 1
print(a ** b) # 출력: 1000
