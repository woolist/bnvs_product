파이썬 교재 - 문자열 자료형

문자열(String)이란?

문자열은 파이썬에서 가장 일반적으로 사용되는 자료형 중 하나입니다. 문자열은 작은 따옴표('')나 큰 따옴표("")로 둘러싸여 있는 문자들의 집합입니다.

예를 들어, 'Hello, World!'이나 "Python"은 모두 문자열입니다.

문자열 생성하기

문자열을 생성하는 방법은 다양합니다.

1. 작은 따옴표('')나 큰 따옴표("")로 둘러싸기

string1 = 'Hello, World!'
string2 = "Python"

2. 문자열 연결하기

string3 = string1 + ' ' + string2

3. 문자열 반복하기

string4 = 'abc' * 3

문자열 다루기

문자열에는 다양한 기능과 메서드가 있어 문자열을 다루기에 매우 편리합니다.

1. 인덱스

text = 'Hello, World!'
print(text[0])  # 'H'

2. 슬라이싱

text = 'Hello, World!'
print(text[7:12])  # 'World'

3. 길이 구하기

text = 'Hello, World!'
length = len(text)
print(length)  # 13

4. 대소문자 변환

text = 'Hello, World!'
print(text.upper())  # 'HELLO, WORLD!'
print(text.lower())  # 'hello, world!'

문자열 포맷팅

문자열 포맷팅은 문자열 안에 변수의 값이나 다른 표현식을 삽입하는 것을 의미합니다.

1. % 연산자를 이용한 포맷팅

name = 'Tom'
age = 25
print("My name is %s and I'm %d years old" % (name, age))

2. format() 메서드를 이용한 포맷팅

name = 'Tom'
age = 25
print("My name is {} and I'm {} years old".format(name, age))

3. f-string을 이용한 포맷팅 (Python 3.6 이상)

name = 'Tom'
age = 25
print(f"My name is {name} and I'm {age} years old")

문자열 관련 함수와 메서드

문자열을 다루기 위해 사용할 수 있는 함수와 메서드들이 있습니다.

1. len() 함수

text = 'Hello, World!'
length = len(text)
print(length)  # 13

2. split() 메서드

text = 'Python is easy'
words = text.split(' ')
print(words)  # ['Python', 'is', 'easy']

3. replace() 메서드

text = 'Hello, World!'
new_text = text.replace('Hello', 'Hi')
print(new_text)  # 'Hi, World!'
