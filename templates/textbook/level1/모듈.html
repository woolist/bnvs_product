파이썬 모듈 교재

모듈이란?

파이썬에서 모듈은 코드의 재사용을 위해 사용되는 파일입니다. 모듈은 함수, 변수, 클래스 등의 정의를 포함할 수 있으며 다른 파이썬 프로그램에서 불러와 사용할 수 있습니다.

모듈의 임포트

다른 파이썬 파일에서 모듈을 사용하기 위해서는 모듈을 임포트해야 합니다. 모듈을 임포트하는 방법은 다음과 같습니다.

import 모듈이름

모듈의 사용

모듈을 임포트했다면 해당 모듈에 정의된 함수, 변수, 클래스 등을 사용할 수 있습니다. 모듈의 사용 예시는 다음과 같습니다.

import 모듈이름
모듈이름.함수이름()
모듈이름.변수이름

모듈 만들기

새로운 모듈을 만들기 위해서는 파이썬 파일을 생성한 후 해당 파일에 함수, 변수, 클래스 등을 정의하면 됩니다. 예를 들어, 다음은 hello.py라는 모듈을 만들고 사용하는 예시입니다.

# hello.py 파일
def say_hello():
    print("Hello, world!")

# main.py 파일
import hello

hello.say_hello()

다른 모듈 가져오기

파이썬에서는 이미 기본으로 제공되는 여러 모듈들을 가져와 사용할 수 있습니다. 다른 모듈을 가져오는 방법은 다음과 같습니다.

import 모듈이름

또는

from 모듈이름 import 함수이름, 변수이름

모듈 문서화

모듈을 만들 때 문서화를 함으로써 다른 사용자들이 모듈을 보다 쉽게 이해하고 사용할 수 있습니다. 모듈 문서화는 다음과 같은 형식으로 작성됩니다.

"""
이 부분에 모듈 내용과 사용 방법을 설명합니다.
"""
