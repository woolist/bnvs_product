파이썬 튜플 자료형

튜플은 변경할 수 없는(immutable) 순서가 있는 자료형입니다. 파이썬에서 튜플은 괄호로 둘러싸여 있으며, 각 요소는 쉼표로 구분됩니다.

튜플 생성하기

튜플을 생성하는 방법은 여러 가지가 있습니다.

1. 괄호를 사용하여 튜플 생성하기


tup = (1, 2, 3)
print(tup)  # 출력: (1, 2, 3)

2. 괄호 없이 쉼표로만 구분하여 튜플 생성하기


tup = 1, 2, 3
print(tup)  # 출력: (1, 2, 3)

튜플의 인덱싱과 슬라이싱

튜플은 리스트와 마찬가지로 인덱싱과 슬라이싱을 지원합니다.

튜플의 요소에 접근하기 위해서는 []를 사용한 인덱싱을 할 수 있습니다.


tup = (1, 2, 3)
print(tup[0])  # 출력: 1

튜플의 일부분을 추출하기 위해서는 슬라이싱을 사용할 수 있습니다.


tup = (1, 2, 3, 4, 5)
print(tup[1:4])  # 출력: (2, 3, 4)

튜플의 메소드

튜플은 변경할 수 없기 때문에 메소드들이 제한적입니다. 주요한 메소드로는 다음과 같은 것들이 있습니다.

  • count(): 지정한 값과 일치하는 요소의 개수를 반환합니다.
  • index(): 지정한 값과 일치하는 첫 번째 요소의 인덱스를 반환합니다.

tup = (1, 2, 2, 3, 4, 5)
print(tup.count(2))  # 출력: 2
print(tup.index(3))  # 출력: 3

튜플의 활용

튜플은 함수의 반환값이 여러 개일 때 유용하게 사용될 수 있습니다.


def get_name_and_age():
    name = "John"
    age = 25
    return name, age

person = get_name_and_age()
print(person)  # 출력: ("John", 25)

name, age = get_name_and_age()
print(name)  # 출력: "John"
print(age)  # 출력: 25

또한, 튜플은 요소의 순서가 중요한 경우에 사용될 수 있습니다.

요약

파이썬 튜플 자료형은 변경할 수 없는 순서가 있는 자료형입니다. 튜플은 괄호로 둘러싸여 있으며, 각 요소는 쉼표로 구분됩니다. 튜플은 인덱싱과 슬라이싱을 지원하며, 일부 메소드들을 활용할 수 있습니다. 또한, 함수의 반환값이 여러 개인 경우나 요소의 순서가 중요한 경우에 튜플은 유용하게 사용될 수 있습니다.
