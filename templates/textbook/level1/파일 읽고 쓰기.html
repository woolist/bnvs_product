파이썬 파일 읽고 쓰기 교재 ============================ 1. 파일 입출력 기본 개념 ------------------------ - 파일 입출력이란 컴퓨터의 디스크 저장소에 있는 파일에 데이터를 읽고 쓰는 것을 말합니다. - 파이썬에서는 `open` 함수를 사용하여 파일을 열고, `read`와 `write` 메서드를 사용하여 데이터를 읽고 씁니다. 2. 파일 열기 ----------- - `open` 함수를 사용하여 파일을 열 수 있습니다. - `open` 함수는 첫 번째 인자로 파일 경로를 지정하고, 두 번째 인자로 파일 열기 모드를 지정합니다. - 파일 열기 모드에는 다음과 같은 옵션이 있습니다. - "r": 읽기 모드로 파일을 엽니다. 파일이 존재하지 않으면 오류가 발생합니다. - "w": 쓰기 모드로 파일을 엽니다. 파일이 존재하지 않으면 새로 생성합니다. 이미 파일이 존재할 경우에는 덮어쓰게 됩니다. - "a": 추가 모드로 파일을 엽니다. 파일이 존재하지 않으면 새로 생성합니다. 이미 파일이 존재할 경우에는 뒤에 내용을 추가합니다. - 예시: `file = open("example.txt", "r")` 3. 파일 읽기 ----------- - `read` 메서드를 사용하여 파일에서 데이터를 읽을 수 있습니다. - `read` 메서드는 파일의 내용 전체를 문자열로 반환합니다. - 예시: `data = file.read()` 4. 파일 쓰기 ----------- - `write` 메서드를 사용하여 파일에 데이터를 씁니다. - `write` 메서드는 인자로 문자열을 받으며, 해당 문자열을 파일에 기록합니다. - 예시: `file.write("Hello, World!")` 5. 파일 닫기 ----------- - 작업이 끝난 파일은 `close` 메서드를 사용하여 닫아야 합니다. - 파일을 열고 작업을 완료한 뒤에는 네트워크 연결과 같이 자원을 해제해야 하기 때문입니다. - 예시: `file.close()` 6. 파일 입출력 예제 ----------------- ``` file = open("example.txt", "w") file.write("Hello, World!") file.close() ``` 7. 연습 문제 ------------ - 파일의 내용을 읽어서 출력하는 간단한 프로그램을 작성해보세요. ```python file = open("example.txt", "r") data = file.read() print(data) file.close() ``` 이렇게 파이썬 파일 읽고 쓰기에 대한 기본 내용을 알아보았습니다. 추가적인 학습이 필요하다면 파이썬 공식 문서나 다양한 개발자 커뮤니티의 자료들을 참고하시면 좋을 것입니다.
