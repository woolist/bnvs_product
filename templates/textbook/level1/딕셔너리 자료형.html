파이썬 딕셔너리 자료형

딕셔너리는 파이썬의 중요한 자료형 중 하나입니다. 딕셔너리는 키(key)와 값(value)의 쌍을 요소로 가지는 자료형입니다. 이를 통해 키를 통해 값을 빠르게 찾을 수 있습니다.

딕셔너리 생성하기

딕셔너리는 중괄호 {}를 사용하여 생성하며, 키와 값은 콜론(:)으로 연결합니다. 예를 들어, 다음과 같이 딕셔너리를 생성할 수 있습니다.

my_dict = {'apple': 1, 'banana': 2, 'cherry': 3}

딕셔너리에서 값 접근하기

딕셔너리에서 값을 접근하려면, 대괄호 [] 안에 키를 넣어 사용합니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

print(my_dict['apple'])  # 출력: 1

딕셔너리에 값 추가하기

딕셔너리에 새로운 키-값 쌍을 추가하려면, 새로운 키를 대괄호 [] 안에 넣고 값을 할당하면 됩니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

my_dict['orange'] = 4
print(my_dict)  # 출력: {'apple': 1, 'banana': 2, 'cherry': 3, 'orange': 4}

딕셔너리에서 값 제거하기

딕셔너리에서 키-값 쌍을 제거하려면, 'del' 키워드를 사용합니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

del my_dict['apple']
print(my_dict)  # 출력: {'banana': 2, 'cherry': 3, 'orange': 4}

딕셔너리의 메소드

딕셔너리는 유용한 메소드를 많이 가지고 있습니다. 예를 들어, keys() 메소드는 딕셔너리의 모든 키를, values() 메소드는 모든 값을 반환합니다.

print(my_dict.keys())    # 출력: dict_keys(['banana', 'cherry', 'orange'])
print(my_dict.values())  # 출력: dict_values([2, 3, 4])
