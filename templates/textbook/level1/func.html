파이썬 함수 교재

함수(function)란?

함수는 어떤 작업을 수행하는 코드의 블록입니다. 이 코드 블록을 재사용하여 여러 번 호출할 수 있습니다. 파이썬에서는 def 키워드를 사용하여 함수를 정의합니다.

함수의 정의

함수를 정의하는 방법은 다음과 같습니다:

def 함수이름(매개변수1, 매개변수2, ...):
    # 함수 내용 작성
    return 결과값

함수의 호출

함수를 호출하는 방법은 다음과 같습니다:

함수이름(인자1, 인자2, ...)

매개변수와 인자

함수에는 매개변수(parameter)라 불리는 변수가 있습니다. 함수를 호출할 때 매개변수에 전달되는 값들을 인자(argument)라고 합니다.

함수의 반환값

함수가 값을 반환(return)하고 싶을 때는 return 키워드를 사용합니다. 반환된 값은 함수를 호출한 곳에서 받을 수 있습니다.

예제

def add(a, b):
    return a + b

result = add(3, 5)
print(result)  # 결과: 8
