파이썬 정규 표현식

1. 정규 표현식이란?

정규 표현식은 특정한 패턴을 가진 문자열을 검색하거나 대체하거나 추출하는데 사용되는 표현식입니다. 파이썬에서는 re 모듈을 사용하여 정규 표현식을 다룰 수 있습니다.

2. 정규 표현식 문법

정규 표현식은 다양한 특수 문자와 메타 문자를 사용하여 패턴을 표현합니다. 아래는 주요한 문법 요소입니다:

  • . : 어떤 한 개의 문자와 매치됩니다.
  • [ ] : 대괄호 사이에 있는 문자들 중 한 개의 문자와 매치됩니다.
  • * : 앞의 문자가 0개 이상일 경우 매치됩니다.
  • + : 앞의 문자가 최소 한 개 이상일 경우 매치됩니다.
  • ? : 앞의 문자가 0개 또는 1개일 경우 매치됩니다.
  • { } : 숫자 또는 범위를 지정하여 일치하는 횟수를 제한할 수 있습니다.
  • | : 둘 중 하나를 의미하며, 각각의 조건 중 매치되는 것을 찾습니다.
  • ^ : 시작 문자를 의미합니다.
  • $ : 종료 문자를 의미합니다.
  • \ : 특수 문자를 일반 문자로 인식합니다. (이스케이프)

3. 정규 표현식 매치 함수

파이썬에서는 re 모듈을 임포트하여 다양한 매치 함수를 사용할 수 있습니다. 주요한 매치 함수는 다음과 같습니다:

  • match : 문자열의 처음부터 정규식과 매치되는지 확인합니다.
  • search : 문자열 전체를 검색하여 정규식과 매치되는 첫 번째 위치를 반환합니다.
  • findall : 문자열에서 정규식과 매치되는 모든 경우를 찾아 리스트로 반환합니다.
  • finditer : 문자열에서 정규식과 매치되는 모든 경우를 반복 가능한 객체로 반환합니다.
  • sub : 문자열에서 정규식에 해당하는 부분을 다른 문자열로 치환합니다.

4. 정규 표현식 실습 예제

예제 1) 이메일 주소 판별하기 


    import re

    email = input("이메일 주소를 입력하세요: ")
    pattern = r"^\w+@[a-zA-Z]+\.[a-zA-Z]+$"

    if re.match(pattern, email):
        print("올바른 이메일 주소입니다.")
    else:
        print("올바르지 않은 이메일 주소입니다.")

예제 2) 전화번호 추출하기 


    import re

    text = "문의 사항은 010-1234-5678로 연락주세요."
    pattern = r"\d{3}-\d{4}-\d{4}"

    phone_numbers = re.findall(pattern, text)
    print(phone_numbers)

예제 3) HTML 태그 제거하기 


    import re

    text = "
안녕하세요, 반갑습니다.

오늘의 날씨는 어떤가요?
"
    pattern = r"<.*?>"

    text_without_tags = re.sub(pattern, "", text)
    print(text_without_tags)
