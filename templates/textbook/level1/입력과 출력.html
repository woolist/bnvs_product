파이썬 입력과 출력

1. 파이썬 입력

파이썬에서 사용자로부터 입력을 받기 위해는 input() 함수를 사용합니다.

name = input("이름을 입력하세요: ")
age = input("나이를 입력하세요: ")

print("안녕하세요, " + name + "님!")
print("당신의 나이는 " + age + "세입니다.")

위의 예제는 사용자로부터 이름과 나이를 입력받아 출력하는 예제입니다. input() 함수는 사용자의 입력을 문자열로 반환합니다. 따라서 입력받은 값을 변수에 저장할 때에는 문자열로 저장해야 합니다.

2. 파이썬 출력

파이썬에서 출력을 위해 print() 함수를 사용합니다.

print("Hello, World!")
print("파이썬 출력 예제")

print() 함수는 괄호 안에 있는 값을 출력합니다. 여러 개의 값을 출력하고 싶다면 쉼표로 구분하여 값을 전달하면 됩니다.

name = "John"
age = 25

print("이름: ", name, "나이: ", age)

위의 예제는 변수에 저장된 값을 출력하는 예제입니다. 쉼표로 구분하여 값을 전달하면 자동으로 공백으로 구분하여 출력됩니다.
