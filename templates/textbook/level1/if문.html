파이썬 if문

파이썬의 if문은 조건에 따라 다른 코드 블록을 실행시킬 수 있는 제어문입니다. if문은 주어진 조건이 참(True)인 경우에만 코드를 실행합니다.

if문의 기본 구조


  if 조건:
      # 조건이 참인 경우 실행되는 코드

if-else문

if-else문은 조건이 참(True)인 경우와 거짓(False)인 경우에 각각 다른 코드 블록을 실행시킬 수 있습니다. 


  if 조건:
      # 조건이 참인 경우 실행되는 코드
  else:
      # 조건이 거짓인 경우 실행되는 코드

if-elif-else문

if-elif-else문은 여러 개의 조건을 순차적으로 검사하여 해당하는 코드 블록을 실행시킬 수 있습니다. 


  if 조건1:
      # 조건1이 참인 경우 실행되는 코드
  elif 조건2:
      # 조건2가 참인 경우 실행되는 코드
  else:
      # 모든 조건이 거짓인 경우 실행되는 코드

if문의 중첩

if문은 다른 if문과 중첩하여 사용할 수 있습니다. 중첩된 if문은 더 복잡한 조건식을 구성할 수 있습니다. 


  if 조건1:
      if 조건2:
          # 조건1과 조건2가 모두 참인 경우 실행되는 코드

논리 연산자

if문에서는 다양한 조건을 결합하여 사용할 수 있는 논리 연산자가 있습니다. 주로 사용되는 논리 연산자는 다음과 같습니다:

  • and - 두 조건이 모두 참일 경우 참
  • or - 두 조건 중 하나라도 참일 경우 참
  • not - 조건의 결과를 반대로 바꿈
