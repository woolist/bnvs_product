파이썬 리스트 자료형

리스트는 파이썬에서 가장 기본적이고 많이 사용하는 자료형 중 하나입니다. 리스트는 여러 개의 값을 저장할 수 있으며, 순서가 있고 수정 가능한 자료형입니다.

리스트 생성

리스트는 대괄호([])를 사용하여 생성할 수 있습니다. 리스트 안에는 각 요소들을 콤마(,)로 구분해서 입력합니다.


# 빈 리스트 생성
empty_list = []

# 숫자로 이루어진 리스트 생성
numbers = [1, 2, 3, 4, 5]

# 문자열로 이루어진 리스트 생성
fruits = ['apple', 'banana', 'orange']

리스트 인덱싱

리스트는 각 요소들의 순서를 가지고 있기 때문에, 인덱스를 사용하여 접근할 수 있습니다. 인덱스는 0부터 시작하며, 대괄호([]) 안에 인덱스 값을 입력합니다.


numbers = [1, 2, 3, 4, 5]

# 첫 번째 요소에 접근
first_number = numbers[0]

# 세 번째 요소에 접근
third_number = numbers[2]

# 마지막 요소에 접근
last_number = numbers[-1]

리스트 슬라이싱

리스트 슬라이싱은 리스트에서 여러 개의 요소를 한 번에 가져오는 방법입니다. 시작 인덱스부터 끝 인덱스 전까지의 요소들을 가져옵니다. 콜론(:)을 사용하여 슬라이싱 범위를 지정합니다.


numbers = [1, 2, 3, 4, 5]

# 두 번째부터 네 번째 요소까지 가져오기
sliced_numbers = numbers[1:4]

# 처음부터 세 번째 요소까지 가져오기
sliced_numbers_2 = numbers[:3]

# 세 번째부터 마지막 요소까지 가져오기
sliced_numbers_3 = numbers[2:]

리스트 연산

리스트는 다양한 연산을 수행할 수 있습니다. 리스트끼리의 덧셈 연산, 곱셈 연산, 수정 연산 등이 가능합니다.


fruits = ['apple', 'banana', 'orange']
numbers = [1, 2, 3, 4, 5]

# 리스트 덧셈 (리스트 합치기)
combined_list = fruits + numbers

# 리스트 곱셈 (리스트 반복)
repeated_list = fruits * 2

# 리스트 수정
fruits[0] = 'grape'

리스트 메서드

리스트에는 다양한 메서드가 내장되어 있어서 리스트를 다룰 때 편리하게 사용할 수 있습니다. 몇 가지 예시를 소개합니다.


fruits = ['apple', 'banana', 'orange']

# 요소 추가
fruits.append('kiwi')

# 요소 삭제
fruits.remove('banana')

# 요소 개수 확인
number_of_fruits = fruits.count('apple')

# 요소 정렬
fruits.sort()

# 리스트 비우기
fruits.clear()

리스트와 반복문

리스트와 반복문을 함께 사용하면 리스트의 각 요소들을 순차적으로 접근하면서 작업할 수 있습니다. 다음은 리스트의 모든 요소를 출력하는 예시입니다.


fruits = ['apple', 'banana', 'orange']

for fruit in fruits:
    print(fruit)
