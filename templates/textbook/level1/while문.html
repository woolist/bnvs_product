파이썬 while문

파이썬에서 while문은 특정 조건이 만족되는 동안 반복적으로 코드 블록을 실행하는 데 사용됩니다. 반복문은 특히 특정 작업을 반복적으로 수행해야 할 때 유용합니다.

while문의 기본 구조


while 조건:
    # 코드 블록

while문은 조건이 참인 동안 코드 블록을 계속 실행합니다. 조건은 참 또는 거짓을 반환하는 표현식이어야 합니다. 조건이 거짓이 되면 while문은 종료됩니다.

예제


count = 0
while count < 5:
    print("Hello, world!")
    count += 1

위의 예제에서는 변수 count를 선언하고, count가 5보다 작은 동안 "Hello, world!"를 출력하고 count를 1씩 증가시키는 코드를 반복적으로 실행합니다. 따라서 "Hello, world!"가 총 5번 출력됩니다.

while문에서의 제어문

while문을 실행하는 도중에 특정 조건을 충족하면 반복문을 중지하거나 건너뛸 수도 있습니다. 이를 위해 break와 continue라는 제어문을 사용할 수 있습니다.

break


while 조건:
    if 조건:
        break
    # 코드 블록

위의 예제에서는 조건을 만족하면 break문이 실행되어 while문을 종료시킵니다. break문을 만나면 while문 내의 나머지 코드 블록은 실행되지 않습니다.

continue


while 조건:
    if 조건:
        continue
    # 코드 블록

위의 예제에서는 조건을 만족하면 continue문이 실행되어 while문의 시작점으로 돌아갑니다. 즉, continue문을 실행하면 while문 내의 나머지 코드 블록은 실행되지 않고 다음 반복으로 넘어갑니다.

정리

위에서 다룬 내용을 정리하자면, 파이썬 while문은 특정 조건이 만족되는 동안 반복적으로 코드를 실행하는 데 사용됩니다. while문은 조건이 참인 동안 계속 반복하며, 조건이 거짓이 되면 종료됩니다. 또한, while문 내에서 break와 continue를 사용하여 반복문을 제어할 수 있습니다.
